Quick Setup Guide

Fast-track your deployment! Distribute your modpack globally and invite players in seconds with this rapid checklist.

1

Create Server

Click Create Server to set up an instance. Mero will automatically download and set up the required JRE and engine files.

2

Install Content

Search and download from the Mods & Plugins tab or drag-and-drop custom mods, resourcepacks, or shaderpacks using the File Manager.

3

Publish Manifest

Under the Dashboard's *Hardware Specs & Sync* card, click Publish. Mero builds a delta-sync manifest of all your active mods and assets.

4

Get Invite Code

Select Mero P2P in Settings, and **Start** the server. Copy the generated MERO- invite code shown in the Dashboard's IP text area.

5

Connect & Sync

Players paste the code into their MeroClient, which automatically delta-syncs mods/packs, starts a local proxy, and bridges connection via UDP P2P!

Pro Tip: You can choose between Playit.gg (simple cloud proxy with custom subdomains) or Mero P2P (direct NAT-traversal tunnel with zero-latency proxying) in the Settings tab.
Premium Asset Warning: The sync feature now fully supports resourcepacks and shaderpacks via the File Manager! Drag-and-drop assets into these directories, and they will automatically synchronize. Note: Extremely large packs may impact P2P sync times for players joining the server.

About the Hoster

Mero Server Hoster is a next-generation lightweight Minecraft server management platform. Designed with a custom Emerald and Black aesthetic, it provides users with premium features like auto-Modrinth resolution, Playit.gg tunneling, and asynchronous mod downloading natively.

Playit.gg Integration

We integrate with the Playit.gg tunneling system to securely expose your server without port forwarding. The backend daemon automatically binds the tunnel agent, establishing a cloud-proxy tunnel and routing traffic directly to your local server.

MeroP2P Client

The MeroClient allows players to seamlessly connect to your server. By pasting your Mero Invite Code, the client automatically extracts the manifest, delta-syncs all client-side mods/packs, and tunnels connection over a secure peer-to-peer UDP bridge.
